Hi McKinley,
 
I apologize for the delayed response; I was out of the office. To provide you with a detailed answer, let’s go back to the algorithm (and I apologize if this is repetitive for you). The main equation for the Metropolis criterion is Equation (9), which varies depending on the "l" value of the fractional particle in the old configuration (o) and the new configuration (n) as described in Equations (10), (11), and (12). 
 
The goal is to obtain a self-consistent determination of Q(N,V,T,l) by using the Wang-Landau method in a Monte Carlo simulation, applying these specific Metropolis criteria. To begin the simulation, you set Q(N,V,T,l) to 1 for all N and l, allowing you to calculate the actual values for the Metropolis criterion in Equations (10) to (12) so that you can run your simulations. 
 
For more details on the workflow of the simulation, I’d suggest checking out our article, especially the section that begins with "As can be seen..." on page 3.
 
At the end of the simulation, once it has converged, you will have the value of Q(N,V,T). This means we only use the values for Q(N,V,T,l=0). For Figure 1, we simply use Q^*(N,V,T) = \Lambda^3 Q(N,V,T), which means that we multiply our value Q(N,V,T) by \Lambda^3 to obtain Q^*(N,V,T) for Argon.
 
For CO_2, we will apply the same logic, but this time we will incorporate the term for a rigid molecular fluid (as we discussed last time). In this case, the Metropolis criterion will change to Equation (13).
 
I hope this clarifies things.
